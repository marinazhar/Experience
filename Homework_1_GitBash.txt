1)  Посмотреть, где я:
   pwd
2)  Создать папку:
   mkdir название_папки
3)  Зайти в папку:
   cd название_папки
4)  Создать 3 папки на одном уровне:
   mkdir название_папки1 название_папки2 название_папки3
    Создать 3 папки друг в друге:
   mkdir -p название_папки/название_папки/название_папки
5)  Зайти в любую папку:
   cd название_папки
   cd название_папки/название_папки
    Зайти в верхнюю папку:
   cd .. 
    Зайти в папку на два уровня выше:
   cd ../..
    Зайти в предыдущую папку:
   cd -
6)  Создать 5 файлов: 2.json  и 3.txt
   touch название_файла.json
   > название файла.json
   touch название_файла.json название_файла.txt
   touch название_файла.txt название_файла2.txt название_файла3.txt
7)  Создать 3 папки:
   mkdir название_папки1 название_папки2 название_папки3
8)  Вывести список содержимого папки:
   ls название_папки
    Вывести список содержимого, включая скрытые папки:
   ls -la название_папки
9)  Открыть любой txt файл:
   cat название_файла.txt
10)  Написать туда что-нибудь, любой текст:
   echo ЛюбойТекст >>название файла.txt
   vim название_файла.txt
     В открывшемся окне нажать кнопку "i", написать любой текст.
11) Сохранить и выйти: 
     нажать кнопку "Esc", затем клавиши "wq" и кнопку "Enter"
12) Выйти из папки на уровень выше:
     cd ..
13) Переместить любые 2 файла, которые вы создали, в любую другую папку:
     mv название_файла.txt название_папки/название файла.txt
     mv название_папки/название_файла.json название_папки2/название_файла.json
14) Скопировать любые 2 файла, которые вы создали, в любую другую папку:
     cp название_файла.json название_папки
     cp название_файла.txt название_файла2.json название_папки
15) Найти файл по имени:
     find название_файла.txt
     find название_папки/название_файла.json
16) Просмотреть содержимое в реальном времени:
     tail -f название_файла.txt
    Просмотреть содержимое файла:
     cat название_файла.txt
    Просмотреть содержимое с номерами строк:
     nl название_файла.txt 
    Просмотреть содержимое в отдельном окне в режиме просмотра: (для выхода нажать клавишу "q")
     less название_файла.txt
    Просмотреть файл с первых 10 строк (с помощью команды -n можно корректировать количество появившихся верхних строк):
     head название_файла.txt
     head -n23 название_файла.txt
    Просмотреть файл с последних 10 строк (с помощью команды - можно корректировать количество появившихся нижних строк):
     tail название_файла.txt
     tail -18 название файла.txt
    Команда grep ищет слово по всему тексту в указанном файле:
     grep слово название_файла.txt
17) Вывести несколько первых строк из текстового файла:
     head название_файла.txt (выведутся первые 10 строк)
     head n-5 название_файла.txt (выведутся первые 5 строк)
18) Вывести несколько последних строк из текстового файла:
     tail название_файла.txt (выведутся последние 10 строк)
     tail -5 название_файла.txt (выведутся последние 5 строк)
19) Просмотреть содержимое длинного файла:
     less название_файла.txt
    Для выхода из окна просмотра нажать клавишу "q";
    Для поиска по тексту нажать "/", ввести искомое слово и нажать "enter";
20) Вывести дату и время:
     date
