1. Создать файл "new.json"
   touch new.json
2. Добавить файл под гит
   git add new.json
3. Закоммитить файл
   commit -m "creare new.json"
4. Отправить файл на внешний GitHub репозиторий
   git push
5. Создать файлы new2.json, new2.xml, new2.txt
   touch new2.json new2.xml new2.txt
6. Добавить файды под гит
   git add new2.json new2.xml new2.txt
7. Закоммитить файлы
   git commit -m "create new2"
8. Отправить файлы на внешнйи GitHub репозиторий
   git push
9. Отредактировать содержание файла "new2.txt"
   vim new2.txt --> i --> "esc" --> ":wq"
10. Отредактировать содержимое файла "new2.json"
    vim new2.json --> i --> "esc" --> ":wq"
11. Добавить и закоммитить "new2.txt" одной строчкой
    git commit -am new2.txt
12. Добавить и закоммитить "new2.json" одной строчкой
    git commit -am new2.json
13. Отправить изменения на внешний репозиторий
    git push
14. Отредактировать содержимое файла "new2.xml"
    vim new2.xml --> i --> "esc" --> ":wq"
15. Добавить и закоммить "new2.xml" одной строчкой
    git commit -am new2.xml
16. Отправить изменения на внешний репозиторий
    git push
17. В веб интерфейсе GitHub создать файл new3.txt
18. Отредактировать в веб интерфейсе содержание файла "new3.txt"
19. Сделать commit changes (сохранить) изменения на веб интерфейсе
20. Синхронизировать внешний и локальный репозитории (слить изменения с внешнего рпозитория)
   git pull
21. Отредактировать в веб интерфейсе содержание файла "new2.json"
